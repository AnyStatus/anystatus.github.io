What is AnyStatus?

AnyStatus is a lightweight desktop application that brings together metrics and events from various sources into one simple dashboard, designed to host various widgets and plugins. These include CI/CD build pipelines, releases, deployment environments, work items, pull requests, servers, docker containers, health checks, local system information, and more.

Builds, Releases, Work Items

View the status of your build pipelines, releases, pull requests and work items on TFS, Azure DevOps, Jenkins, TeamCity, AppVeyor

Health Checks

Monitor remote servers, web services, application services, network interfaces, database connections and more

Metrics

Current view of CPU usage, Performance Counters, SQL Queries and other metrics on remote and local computer

Folders

Organize widgets in an unlimited folder structure, events and statuses bubble up and the overall status can be viewed even when the tree is collapsed

Statuses

View the status of your development environment in a simple minimalist dashboard with customizable status indicators

Remote Control

Start or stop deployments and build pipelines, control remote services, open in browser and more

Notifications

Get notified when builds complete or when services are down

Triggers

Trigger a command-line action or custom notifications on all or specific status changes

Standalone

No server is required

Lightweight

AnyStatus runs in the background and does not have a significant impact on the performance or startup of Windows

Extensible

Develop your own widgets using C#, VB.NET, PowerShell or Batch scripts
