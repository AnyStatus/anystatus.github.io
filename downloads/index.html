AnyStatus Desktop

AnyStatus is a lightweight desktop application designed to bring you metrics and events from various sources.

To install the application, click the download button, then start the downloaded file and follow on-screen instructions.

System Requirements

  • Windows 7 or above.
  • .NET Framework 4.6 or above.
