Latest Release

AnyStatus brings together metrics and events from various sources into one place. AnyStatus is a lightweight application designed to host various widgets and plugins. These include CI/CD builds, releases, deployment environments, work items, pull requests, docker containers, health checks, system information, and more.

To install the application, click the download button, then start the downloaded file and follow on-screen instructions.

System Requirements

  • Windows 7 or above.
  • .NET Framework 4.6 or above.
