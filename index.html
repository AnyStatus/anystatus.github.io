What is AnyStatus?

AnyStatus is a lightweight desktop notifications and dashboard app designed to host various widgets. These include builds, releases, deployment environments, work items, health checks and metrics. It runs in the background and notifies you whenever something important happens

Builds, Releases, Work Items

View the status of your builds, releases and work items on TFS, Azure DevOps, Jenkins, TeamCity, AppVeyor

Health Checks

Monitor the health and availability of remote servers, Windows services, network, database connections and more

Metrics

View the CPU usage, Performance Counters, SQL Queries and other metrics on remote and local computers

Folders

Organize your widgets in an unlimited folder structure. Events and statuses bubble up so the overall status can be viewed when the tree is collapsed

Statuses

View the status of your development environment in a simple dashboard with customizable status indicators

Remote Control

Start builds, control remote Windows services, open web pages and more

Notifications

Get notified when builds complete or when services are down

Triggers

Trigger a command-line action or custom notifications on all or specific status changes

Standalone

Lightweight

Extensible
