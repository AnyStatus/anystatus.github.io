Remote Control

Desktop Notifications

Get notified about status changes when builds are queued or when services are down.

Lightweight

AnyStatus for Visual Studio is a lightweight extension for monitoring processes and services on the back end of your development workflow. It lets you monitor build and integration services like AppVeyor, Jenkins, TeamCity, or TFS, local Windows services, HTTP status, TCP ports, and more. Colored service health indicators are displayed in a tab within Visual Studio. For more information visit Visual Studio Marketplace.
