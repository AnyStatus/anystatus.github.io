Dashboard controls that keep you organized

Builds & Releases

AnyStatus works with Team Foundation Server, Visual Studio Team Services, Jenkins, TeamCity and AppVeyor.

Health Checks

Monitor the health and availablility of remote computers, Windows services, HTTP servers, databases and more.

Metrics

View CPU usage, Performance Counters, SQL Queries and other metrics on remote and local computers.

Notifications

Get notified about status changes when builds complete or when services are down.

Folders

Organize your widgets in an unlimited folder structure. Events and statuses bubble up so the overall status can be viewed when the tree is collapsed.

Status Indicators

View the status of your development environment in a simple dashboard with customizable status indicators.

Remote Control

Start builds, control remote Windows services, open web pages and more.

Desktop Notifications

Get notified when builds complete or when services are down.

Extensible

Develop you own widgets using C#, VB.NET and PowerShell.

Lightweight

AnyStatus runs in the background and does not have a significant impact on the performance or startup of Visual Studio.

Open Source
