Overview

AnyStatus is a lightweight desktop application that brings together metrics and events from various sources into one place. Examples include Jenkins, Azure DevOps, AppVeyor, TeamCity, GitHub, Health Checks, OS Metrics, Network, Databases, and more. It's also available as Visual Studio extension.

Jenkins Build Monitor

Using AnyStatus, you can view the status and progress of jobs on one or more Jenkins servers. It displays an updated view automatically every few seconds using Jenkins REST API. You can also trigger jobs remotely.

Jenkins Views

AnyStatus supports Jenkins Views which is a list of jobs that you can configure on your Jenkins server. Whenever a job is added or removed from the view, AnyStatus will automatically add or remove it from the dashboard.

Desktop Notifications

By default, AnyStatus will show a desktop notification whenever a job is triggered, complete or fail. Or, you can customize notifications using AnyStatus Triggers to show notifications only in certain conditions such as when a job fails.

Getting Started

Download and install AnyStatus from our Website, Microsoft Store or Visual Studio Marketplace.

To learn more about getting started with AnyStatus, check out the Getting Started guide.

Add a new widget by clicking the Add Widget button on the toolbar menu. Select the DevOps category and then Jenkins Job. Copy and paste the Jenkins Job URL address to the URL field and click Save.
